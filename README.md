# 📝 Post Blog App

A simple blog application built using **Node.js**, **Express**, and **EJS**.

## 🚀 Features
- Create, Read, Update, and Delete (CRUD) blog posts  
- Clean and responsive UI with Bootstrap  
- Data fetched from an external API  

## 🛠️ Technologies Used
- Node.js  
- Express.js  
- EJS  
- Axios  
- Bootstrap
